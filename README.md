* coreutils

### Low-memory versions

* [BusyBox](https://www.busybox.net)

* dash (Debian Almquist shell)
* ksh (Korn shell)
* mksh (MirBSD ksh)

### Other Unix shells

* rc (Plan 9 shell)
* csh (C Shell)
* tcsh (TENEX-inspired C shell)
* fish (Friendly interactive shell)

### Windows/DOS shells

* command.com (FreeDOS)
* command.com (MS-DOS shell)
* cmd.exe (Microsoft Windows shell)
* pwsh (Microsoft PowerShell)
